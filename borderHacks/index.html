The Purpose

When browsing the internet, security is a top priority. How much do you really know about the websites you visit? We built this tool to see how safe a website is based on certain criteria, as well as pull information from a site and explain it in a easy to understand way.

Our tool will take a user entered URL and scrub information on it from a python scanner. We want to promote web safety as well as make the general public more aware of threats you can run into online. This is a topic we hold in high regard, and see a major importance for it.
